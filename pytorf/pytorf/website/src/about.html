About pytorf

Overview

pytorf is a Python package designed for reading NetCDF files into Pandas DataFrames. It provides convenient functionalities to extract, process, and summarize observational data from NetCDF files.

Features

  • Easy integration with Pandas for data manipulation.
  • Support for various observational data formats.
  • Efficient data extraction and processing capabilities.

Usage: obs_read_nc

The obs_read_nc function can be imported and called with the appropriate parameters. Here is a basic example:

from pytorf.obs_read_nc import obs_read_nc

# Assuming you have an index DataFrame ready:
data = obs_read_nc(index, categories="flask", solar_time=False, as_list=False, verbose=True)

Usage: obs_summary

The new obs_summary function processes directories containing observational files, assigns each file an ID and sector based on predefined categories, and extracts additional metadata. Hereâ€™s an example usage:

from pytorf.obs_summary import obs_summary

# Provide the path to the directory with observational files.
index = obs_summary('path/to/obs', verbose=True)

This function will output a DataFrame containing file names, categories, and metadata such as file sequence numbers and a computed 'agl' value from the file name. If an output file is specified via the out parameter, the DataFrame is also saved as a CSV.
